### 🧮 Calculadora em C

Uma calculadora simples desenvolvida para praticar:

* Estruturas condicionais
* Funções
* Organização de código
* Entrada e saída de dados
* Operações matemáticas básicas

**Operações disponíveis**

* Soma
* Subtração
* Multiplicação
* Divisão

📂 Caminho do projeto:

```
Calculadora-C/main.c
```
