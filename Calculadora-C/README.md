# 🧮 Calculadora em C

Projeto desenvolvido em **linguagem C** com o objetivo de praticar conceitos fundamentais da programação.

O programa funciona no terminal e permite realizar operações matemáticas básicas.

---

# 🚀 Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão

O usuário escolhe a operação através de um menu interativo.

---

# 📚 Conceitos praticados

Este projeto foi desenvolvido para praticar:

* Funções
* Ponteiros
* Estruturas condicionais (`if`, `switch`)
* Laço de repetição (`do while`)
* Entrada e saída de dados (`scanf`, `printf`)

---

# ▶️ Como executar

Compile o programa:

```
gcc main.c -o calculadora
```

Execute:

```
./calculadora
```
