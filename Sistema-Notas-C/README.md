# 🎓 Sistema de Notas em C

Programa desenvolvido em linguagem **C** para gerenciar notas de alunos através de um menu interativo no terminal.

## 🚀 Funcionalidades

* Cadastrar nota
* Mostrar todas as notas cadastradas
* Calcular média da turma
* Mostrar maior nota

## 📚 Conceitos praticados

* Vetores
* Funções
* Ponteiros
* Estruturas condicionais
* Laços de repetição
* Organização de código

## ▶️ Como executar

Compile o programa:

```
gcc main.c -o sistema_notas
```

Execute:

```
./sistema_notas
```
